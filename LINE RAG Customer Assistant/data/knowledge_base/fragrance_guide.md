# คู่มือเลือกกลิ่น & วิธีใช้ก้านไม้หอม

## เลือกกลิ่นตามความต้องการ
- อยากนอนหลับง่าย / ผ่อนคลาย / ลดเครียด → แนะนำ **Calm Down (ลาเวนเดอร์)**
- อยากสดชื่น มีพลัง ตื่นตัว ช่วงเช้าหรือทำงาน → แนะนำ **Main Character (เบอร์กามอต-ส้ม)**
- อยากให้ห้องดูหรู มีกลิ่นอบอวลแบบมีระดับ ต้อนรับแขก → แนะนำ **Soft Drama (มะลิ)**

## เลือกกลิ่นตามห้อง
- ห้องนอน: Calm Down
- โต๊ะทำงาน / ห้องนั่งเล่นตอนกลางวัน: Main Character
- ห้องรับแขก / มุมแต่งตัว: Soft Drama
- คอนโดห้องสตูดิโอ (ห้องเดียว): เลือกตามช่วงเวลาที่อยู่บ้านบ่อยที่สุด หรือสลับรีฟิลได้

## วิธีใช้ก้านไม้หอม
1. เปิดฝาขวด แล้วเสียบก้านไม้ทั้งหมดลงไป
2. รอประมาณ 1-2 ชั่วโมงให้ก้านดูดน้ำหอมขึ้นมา
3. กลับก้าน (พลิกด้านที่ชุ่มขึ้นด้านบน) สัปดาห์ละ 1-2 ครั้งเพื่อให้กลิ่นกระจายดีและสม่ำเสมอ

## ปรับความแรงของกลิ่น
- อยากกลิ่นแรงขึ้น: เสียบก้านเยอะขึ้น และกลับก้านบ่อยขึ้น
- อยากกลิ่นเบาลง: ลดจำนวนก้าน
- ห้องใหญ่/มีลมแอร์แรง น้ำหอมจะระเหยเร็วขึ้น

## ข้อควรระวัง (ทั่วไป)
- วางบนพื้นผิวเรียบ ห่างจากมือเด็กและสัตว์เลี้ยง
- หลีกเลี่ยงการวางโดนแสงแดดโดยตรงหรือใกล้ความร้อน
- ระวังน้ำหอมหกบนผิวไม้/พลาสติก/ผ้า อาจทิ้งรอยได้
- ถ้าน้ำหอมเข้าตาหรือผิวหนังแล้วระคายเคือง ให้ล้างด้วยน้ำสะอาด
- หากมีอาการแพ้ ตั้งครรภ์ หรือมีสัตว์เลี้ยงที่อ่อนไหวต่อกลิ่น ควรปรึกษาผู้เชี่ยวชาญก่อนใช้

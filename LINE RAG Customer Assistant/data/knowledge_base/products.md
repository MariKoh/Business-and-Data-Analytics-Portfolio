# MooM HoM — Product Catalog (ก้านไม้หอม / Reed Diffuser)

แบรนด์ MooM HoM "Your Moment. Designed." ก้านไม้หอมสำหรับคนเมืองที่อยู่คอนโด
ทุกขวดออกแบบกลิ่นให้เข้ากับช่วงเวลาและอารมณ์ของคุณ

## Calm Down (กลิ่นลาเวนเดอร์ / Lavender)
- คอนเซ็ปต์: กลิ่นสำหรับผ่อนคลายก่อนนอน ลดความเครียดหลังเลิกงาน
- โทนกลิ่น: ลาเวนเดอร์นุ่ม ๆ อบอุ่น สบายจมูก ไม่ฉุน
- เหมาะกับ: ห้องนอน มุมอ่านหนังสือ ช่วงเวลากลางคืน
- ขนาด: 100 ml
- ราคา: 390 บาท
- กลิ่นอยู่ได้ประมาณ: 6-8 สัปดาห์ (ขึ้นกับขนาดห้องและจำนวนก้าน)

## Main Character (กลิ่นเบอร์กามอต-ส้ม / Bergamot-Orange)
- คอนเซ็ปต์: กลิ่นสดชื่นเพิ่มพลัง สำหรับวันที่อยากเป็นตัวเอกของวัน
- โทนกลิ่น: ส้มเบอร์กามอตสดใส มีความซิตรัส อบอุ่นนิด ๆ
- เหมาะกับ: ห้องนั่งเล่น โต๊ะทำงาน ช่วงเช้า-กลางวัน
- ขนาด: 100 ml
- ราคา: 390 บาท
- กลิ่นอยู่ได้ประมาณ: 6-8 สัปดาห์

## Soft Drama (กลิ่นมะลิ / Jasmine)
- คอนเซ็ปต์: กลิ่นหรูละมุน มีความเป็นผู้หญิง อบอวลแบบมีระดับ
- โทนกลิ่น: มะลิหวานละมุน ฟลอรัล มีมิติ
- เหมาะกับ: ห้องนั่งเล่น ห้องรับแขก มุมแต่งตัว
- ขนาด: 100 ml
- ราคา: 390 บาท
- กลิ่นอยู่ได้ประมาณ: 6-8 สัปดาห์
